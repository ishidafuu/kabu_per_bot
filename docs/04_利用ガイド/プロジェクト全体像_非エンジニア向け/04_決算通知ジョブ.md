# 4. 決算通知ジョブ

## 目的

決算前後の重要日程を先に把握し、調査・準備の遅れを防ぐための通知です。

## 2種類の通知

- `今週決算`: 来週予定分を週次でまとめて通知
- `明日決算`: 翌日予定分を日次で通知

## 実務での使い分け

- `今週決算`: 週次の優先順位づけに使う
- `明日決算`: 前日の最終確認に使う

## 対象範囲

通知対象はウォッチリスト登録銘柄のみです。  
ウォッチリストに無い銘柄の決算は通知されません。

## 運用上の確認ポイント

- 決算カレンダーが最新化されているか
- ウォッチリスト更新直後に通知対象が変わっていないか
- 通知時刻に合わせたチームの確認体制があるか
