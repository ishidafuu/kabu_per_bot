# 5. IR・SNS・AI注目通知

## 何を検知するか

割安判定とは別に、企業発信の更新を検知します。

- `IR更新`: 企業IRページの新着
- `SNS注目`: X（旧Twitter）投稿の更新
- `AI注目`: 収集情報をもとにした注目ポイント通知

## 通知の意図

- ファンダメンタルズに影響する新情報を早めに把握する
- 数値判定だけでは見えない変化を補足する

## 重複通知の抑制

同じURL・同じ投稿IDは重複送信しない設計です。  
同一情報が再掲されても通知が連投されにくくなっています。

## AI注目の有効条件

以下の両方が必要です。

- 全体設定でAI通知が有効
- 銘柄設定で `ai_enabled=true`

どちらか一方が無効なら、AI注目は送信されません。

## 運用時の注意

- X APIトークン期限切れはSNS通知停止の主因
- IR URL不備はIR更新の未検知につながる
- AI通知は補助情報として扱い、最終判断は人が行う
